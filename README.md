# project-calculator
This is a project from "Odin Project" in which we will be making a working Calculator using HTML, CSS and JavaScript.

### Goals for this project: 
- Write Efficient code 
- Better pseudo code 
- Writing Clean Code
- Better UI design 
- Better HTML and CSS

### Features: 
* A calculator UI (user interface)
* Basic math operations like Addition, Subtraction, Multiplication and Division 
* Preforms more than one math operation at once 
* Decimal values work unless it's more than one decimal point value 
* Handles Errors with division by 0
